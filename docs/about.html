This minisite hosts the MA Thesis project of Tommaso Zennaro, called Type Metrics.

This project was developed between October 2013 and January 2014, and is the sum of my MA course in Graphic & Editorial Design at Isia in Urbino, Italy.
I chose to go the hard way for this project so I studied a new language.
In this way I was able to develop a tool that would be helpful for my future projects.

I chose JavaScript and in particular the library d3.js, since the thing I love the most doing is data visualization. Learning and developing something from scratch for this project was something I really wanted to do for a long time, and I am thankful for the chance I had in doing this.

The libraries I used to build the graphs are: d3.js for the main graphic visualization, regression.js for generating the points of the linear regression line, underscore.js to do some array magic & some pure JavaScript.

This project wouldn’t have been possible without the help of my supervisor Luciano Perondi, my co-supervisor Valentina Rachiele, the dataset provided by Roberto Arista and some help from the StackOverflow community, especially Lars Kothoff.
