This minisite hosts the MA Thesis project of Tommaso Zennaro, called Type Metrics.

The main aim of this website is to show an environment that visualizes several data about typography. A former graduated student from Isia in Urbino, Roberto Arista, built a Python script capable of analyzing and obtaining informations out of fonts.

Data such as x-height, contrast, weight etc., for a total of 23 variables, out of 411 fonts (as of now) were saved in a database. By building a JavaScript environment and by using the library d3.js, I made it possible to visualize all those data.

These visualizations are useful for a new way of thinking about fonts and typography. They clearly show, also, many correlations and recurring habits among some aspects about fonts, font styles and font families.
